## Android

- instalar android
  
        npm install @capacitor/android

- agregar android
  
        npx cap add android

- sincronizar
  
        npx cap sync

- abrir
  
        npx cap open android

## QrCode

- Scanner

        npm install @capacitor-mlkit/barcode-scanning

- Generador
  
        npm install angularx-qrcode --save
